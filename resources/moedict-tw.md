# Moedict 萌典

**URL:** https://www.moedict.tw/

**Language:** Chinese

**Region:** Taiwan

## Summary

Moedict is an open digital dictionary project built around Taiwan's Ministry of Education dictionary data and related open lexical resources. The public site provides searchable entries for Mandarin, Taiwanese, and Hakka, with cross-references, pronunciation and romanization information where available, and multilingual lookup support.

For Chinese studies, it is useful as a quick reference for Taiwan usage, variant forms, pronunciations, and dictionary definitions. The site also links to downloadable desktop versions and mobile apps, making it practical for both online consultation and offline language work.
