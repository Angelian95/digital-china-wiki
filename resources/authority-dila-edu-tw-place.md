# DILA Place Authority Database 地名規範資料庫

**URL:** https://authority.dila.edu.tw/place/

**Language:** Chinese

**Region:** Taiwan

## Summary

The DILA Place Authority Database is a Buddhist studies gazetteer and authority-control resource from Dharma Drum Institute of Liberal Arts. Its search interface supports lookup by place name and authority ID, with advanced options for administrative region and category.

Records can include normalized place names, authority codes, historical administrative classifications, latitude and longitude, KML/map links, notes, and references to Buddhist textual occurrences. The database is useful for resolving place names in Buddhist texts, mapping Buddhist sites, and connecting DILA/CBETA references to stable geographic entities.
